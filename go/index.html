API Endpoints

Test Endpoints
  • POST /api/test/run-all - Run all tests
  • POST /api/test/date-variations - Date variations test
  • POST /api/test/expiration-calc - Expiration calculation
  • POST /api/test/urgency-levels - Urgency levels
  • POST /api/test/deposit-calc - Deposit calculation
  • POST /api/test/pricing-rates - Pricing & rates
  • POST /api/test/email-template - Email template
  • POST /api/test/weather-forecast - Weather forecast
  • POST /api/test/form-validation - Form validation
System Endpoints
  • GET /api/health - System health check
  • GET /api/health/ready - Readiness check
  • GET /api/health/live - Liveness check
  • POST /api/email/quote - Send quote email
  • POST /api/email/quote/preview - Preview quote email
  • POST /api/estimate - Calculate estimate
  • GET /api/estimate/special-dates - Get special dates
  • GET /api/quote/pdf?token=... - Download PDF quote (public)
  • POST /api/quote/regenerate - Regenerate expired quote (public)
