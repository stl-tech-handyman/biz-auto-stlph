## Что нужно сделать СЕЙЧАС:

### 1. Остановите текущий сервер
Найдите окно PowerShell где запущен сервер и нажмите **Ctrl+C**

### 2. Запустите сервер заново
```powershell
cd go
.\start-local.ps1
```

### 3. Откройте Swagger UI
После запуска откройте в браузере:
```
http://localhost:8080/swagger
```

## Если не работает:

1. Убедитесь, что сервер запущен из папки `go/`
2. Проверьте логи сервера на наличие ошибок
3. Откройте консоль браузера (F12) и проверьте ошибки
4. Убедитесь, что файл существует: `go/docs/api/openapi-ru.yaml`
